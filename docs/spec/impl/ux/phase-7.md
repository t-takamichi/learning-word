## 2. 動作検証手順
1.  クイズをわざと5回連続で正解し、コンボ演出が2.5秒以内で自動で終了することを確認します。
2.  セッション完了時に、音と画面演出を確認し、完了後の「次のおすすめセットへ進む」ボタンで次の単語セットのクイズがシームレスに開始されることを確認します。
3.  Mac / iPhone の「設定 ➡️ アクセシビリティ ➡️ 動作 ➡️ 視覚効果を減らす」をONにした状態でクイズを行い、アニメーションが適切に抑制されることを確認します。
