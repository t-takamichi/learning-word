# 実装計画書 - フェーズ UX-5: Organisms (主要ブロック)

本フェーズでは、Molecules / Atoms を組み合わせてドメインビジネスロジックや API、フックと結線する「画面の主要ブロック（Organisms）」の構築と移行を行います。

## 2. 動作検証手順
1.  各 Organism 単体で `yarn typecheck` を行い、エラーがないことを確認します。
2.  `WordList` の英単語入力フォームにキー入力し、デバウンス（300ms）を挟んでサジェストAPI（`/api/dictionary/search`）が正常にコールされるかネットワークタブで監視します。
3.  削除用 `ConfirmModal` を表示し、マイクロコピーが応援形（「おわかれする？」等）になっていることを目視確認します。
