# Quality Bar — ディズニーPMとしての品質判断

親: [README.md](./README.md)

> 「あなたはディズニーのPMとして、このレベルのサイトで良いのかしっかり判断しなさい」への回答。
> ディズニーが体験づくりで使う思想を、この単語帳の合否基準に翻訳する。

---

## 1. 判断に使うディズニーの4思想

| 思想 | 意味 | このアプリへの適用 |
|------|------|-------------------|
| **Plussing（プラッシング）** | 「もう一手間」で体験を足し算する | 正解にただ○を出すのではなく、音＋弾み＋粒＋褒め言葉を"重ねる" |
| **Theming（世界観の一貫性）** | 端から端まで同じ物語 | ピンクの世界観・マスコット・やさしい言葉を全画面で統一。ロゴから完了画面まで途切れさせない |
| **Emotional Arc（感情曲線）** | 入場→高揚→クライマックス→余韻を設計 | 開く=ワクワク → 学習=小さな達成の連続 → 完了=クライマックスの祝福 → 「また明日」への余韻 |
| **Attention to Detail** | ゲストは細部で"愛されている"と感じる | 角丸・影・間・マイクロコピー・失敗時のやさしさ。細部の質＝愛情の量 |

---

## 2. 感情曲線（Emotional Arc）の設計

```
高揚 ▲                                   ★完了の祝福
     │                          🔥コンボ  ／╲
     │            ✨正解  ✨正解 ／      ／  ╲ 余韻「また明日」
     │   😊開く   ／╲   ／╲   ／                ＼
     │  ／‾‾‾‾‾‾╱  ╲_／  ╲_／                    ＼___
     └────────────────────────────────────────────────▶ 時間
        入場       学習（小さな達成の連続）        クライマックス   退場
```

- **谷を作らない**：Again（不正解）で感情を下げすぎない。すぐ次の小さな成功へ戻す。
- **クライマックスを必ず用意**：完了画面が"ごほうびのピーク"。ここが弱いと全体が締まらない。
- **良い余韻**：最後に「また来たい」と思わせる一言／次アクションを置く。

## 3. 現行（黒ベース）への PM 判定

**判定：❌ 現状のままではNG（機能は合格・体験は不合格）**

| 観点 | 現状 | 判定 |
|------|------|------|
| 機能の正しさ | 反転・自己評価・音声・リスト等は動作 | ✅ |
| 対象への共感 | 黒×白×グレーは対象（20代女性）に響かない | ❌ |
| ごほうび設計 | 正解しても無反応。感情が動かない | ❌ |
| 世界観 | 無機質。相棒も物語もない | ❌ |
| 細部の愛 | 機能的だが"愛されている感"が薄い | ❌ |

→ **結論**：機能はそのままに、体験（色・演出・音・言葉）を作り込めば合格ラインに乗る。本仕様がその設計。

## 4. 合否チェックリスト（このバーを超えたらGO）

体験レビュー時、以下を **ゲストの気持ち** で確認する。

### Delight（心地よさ）
- [ ] 開いた瞬間「かわいい／やさしい」と感じるか
- [ ] 触っていて気持ちいいか（ぷっくりボタン・なめらかな反転）

### Reward（報われ）
- [ ] 正解のたびに"うれしい"が返ってくるか（音・アニメ・言葉）
- [ ] 失敗しても嫌な気持ちにならないか（責めないか）
- [ ] 完了に"クライマックス"があるか

### Consistency（世界観）
- [ ] 全画面が同じトーンか（浮いた画面がない）
- [ ] 言葉がすべて"やさしい応援形"か

### Craft（細部）
- [ ] 角丸・影・余白・間に不自然さがないか
- [ ] 空状態・ローディング・エラーまで手を抜いていないか

### Respect（配慮）
- [ ] 片手・3秒で使えるか（44px・迷いなし）
- [ ] うるさすぎないか（過剰演出でないか、`reduced-motion`／ミュート可）

> **1つでも❌が残る間は"まだPlussingの余地あり"** として磨く。ただし P4（うるさくしない）と天秤にかけ、足しすぎない。

---

## 5. やりすぎ防止（Anti-goal）

ディズニー品質＝過剰装飾ではない。**上品な余白と間**こそ高級感。

- 🚫 常時アニメが動き続ける／音が鳴りっぱなし
- 🚫 派手すぎる原色・グラデの多用で可読性を損なう
- 🚫 祝福が長すぎて次の操作を待たせる（完了演出は2.5秒以内に収束）
- 🚫 ゲーミフィケーションを盛りすぎて学習の邪魔をする

**「引き算された上質さ」＞「盛り盛りの賑やかさ」**
