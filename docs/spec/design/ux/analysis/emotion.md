## 1. 感情曲線マップ（現状）

| 瞬間 | 期待する感情 | 現状の感情価 | 根拠 |
|------|-------------|:---:|------|
| 起動・入場 | ワクワク | **−** | 黒×白×グレーで第一印象が冷たい。ときめきがない |
| カードをめくる | 期待 | **0** | Y軸フリップなどの演出は控えめ。可もなく不可もなく |
| 正解（Good） | うれしい・報われ | **−** | 無反応。音・アニメ・言葉が皆無（最大の谷） |
| 不正解（Again） | 落ち込まない | **−** | ❌記号＋赤で"間違えた・叱られた"印象 |
| 連続正解 | 高揚 | **0** | ストリークの概念が存在しない（加点機会の損失） |
| セッション完了 | 達成・クライマックス | **0〜−** | 静的🎉のみ。締めが弱くピークが立たない |
| 退場 | また来たい | **−** | 余韻・次の楽しみ・成長実感がない |

## 2. 感情曲線（現状の形）

```
高揚 ▲
     │
   0 ┤━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  ほぼフラット〜下降
     │  ＼            ＼(正解も無反応)        ＼(完了も静か)
 低下 ▼   入場冷たい     谷                    余韻なし
     └────────────────────────────────────────────▶ 時間
```

→ **山（ごほうびのピーク）が一つも無い。** 特に「正解の瞬間」という最も報われるべき点が谷になっている。

## 3. 最優先で埋めるべき谷（Top 3）

1. **正解の谷（最重要）**: Goodに音×視覚×言葉を即時付与 → 曲線に小さな山を連続で作る
2. **完了の谷**: 静的🎉 → 祝福演出＋サマリー＋マスコットでクライマックスを作る
3. **入場の谷**: 黒→ピンクの世界観で第一印象を「ときめき」に変える

## 4. モチベーション機構チェック（現状）

- [ ] 正解に即時フィードバック（音・視覚・言葉） → **無**
- [x] 失敗が罰になっていない → **✕（❌＋赤で罰的）**
- [x] 進捗・ゴールが見える → **△（あるが0始まり・励ましなし）**
- [ ] 継続を促す仕掛け（ストリーク/デイリー） → **無**
- [ ] 完了に"ごほうびのピーク" → **無（静的絵文字のみ）**
- [ ] 世界観／相棒（マスコット） → **無**

## 5. 診断まとめ

現状は「**淡々と正解を記録するだけ**」の体験で、感情を動かす仕掛けがほぼ存在しない。
学習アプリの継続を支える"報われ体験"が構造的に欠落しており、
**機能は成立するが、続けたくなる理由が設計に無い**状態。

→ 改善の主戦場は「正解の瞬間」。ここに山を作ることが最大インパクト。
