# UI/UX分析 - STEP 3: 感情曲線・モチベーション診断

本ドキュメントは、「アプリ起動 → 学習開始 → カードめくり → 回答判定 → セッション完了 → 退場」という学習体験のジャーニーにおけるユーザーの感情起伏（感情曲線）と、学習継続を支えるモチベーション設計の現状を診断したものです。

---

## 1. 感情曲線マップ

学習ジャーニーの各瞬間におけるユーザーの感情価（＋ / 0 / −）とその根拠です。

```
(感情価)
  +5 |   ▲(起動)                                 ▲(完了)
     |  /  \                                   /  \
     | /    \     ▲(正解/コンボ)               /    \
   0 |/      \---/-----\---------------------/      \_______  ▼(退場: 感情の谷)
     |                  \                   /
  -5 |                   ▼(不正解)         /
     +---------------------------------------------------------> (時間軸)
       入場   めくり    Good     Again    コンボ   完了     退場
```

| 瞬間 | 期待する感情 | 現状の感情価 | 判定の根拠 |
|:---|:---|:---:|:---|
| **1. 起動・入場** | ワクワク、癒やし | **＋** | 温かみのある背景グラデーションとマスコット「ベリーちゃん」のイラスト、丸みを帯びたぷっくりしたUIによって、第一印象の可愛らしさと親しみやすさが確立されている。 |
| **2. カードをめくる** | 好奇心、期待 | **0** | CSS 3Dアニメーションによる反転と、シュッとカードが裏返るフリップ音があるが、操作自体はタップのみであり、ドラッグやフリックなど感覚的な「気持ちよさ」は平均的。 |
| **3. 正解（Good）** | 達成感、喜び | **＋** | 三角波の軽快なピコーン音（`correct`）、デバイスの瞬間バイブレーション、画面のキラキラ（スパークル）エフェクトが即座に同期して発動し、心地よい脳内報酬になっている。 |
| **4. 不正解（Again）** | 挫折感のな化 | **0** | 赤ではなく優しいコーラル色を採用し、音も低音で優しい滑らかなスライド音（`again`）に抑えられている。ブザー音などのペナルティ感がなく、挫折しにくい。 |
| **5. 連続正解（コンボ）** | 高揚感、リズム | **＋** | コンボ数がインジケータで蓄積され、一定コンボ（5回以上など）で画面いっぱいにキラキラとベリーちゃんが祝福する `CelebrationOverlay` が作動。単調な作業に変化をつけている。 |
| **6. セッション完了** | クライマックス、達成 | **＋** | 明るいアルペジオ音のファンファーレ、王冠をかぶったベリーちゃん、リザルト画面での華やかな祝福演出により、強い学習達成感（ピーク体験）が得られる。 |
| **7. 退場（学習終了）** | 余韻、また来たい | **−** | **【感情の谷】** 完了画面を閉じると、次の学習目標（「次はこのセットをやってみよう」「今日のノルマ完了！お疲れ様」等）の提示がなく、単に「セット一覧に戻る」ボタンが置かれているのみ。達成感の直後に体験がプツリと途切れ、次に何をしていいか分からなくなる。 |

## 2. モチベーション機構チェックリスト

体験設計（UX PM）の視点（モチベーションデザイン）での実装状況です。

*   **[クリア] 正解への即時フィードバック**
    *   音・振動・視覚（スパークル）の三位一体で、タップの瞬間に即座に「ごほうび」が与えられている。
*   **[クリア] 失敗が罰になっていない設計**
    *   Againを選択しても責められない世界観（やさしい音、コーラル系のカラー）。
*   **[クリア] 進捗・ゴールの可視化**
    *   セッションヘッダーの進捗インジケータ（すすんだ割合）が学習中に常に確認できる。
*   **[クリア] 世界観と相棒（マスコット）の存在**
    *   「ベリーちゃん」が随所に登場し、一緒に学んでいる感覚（相棒感）がある。
*   **[一部不足] 継続を促す仕掛け**
    *   ストリーク（連続学習日数）のバッジや炎マークはあるが、次の学習ノルマや「今日まだ勉強していないよ」といったリマインドの体験が弱い。
*   **[一部不足] 完了後の次アクションへのシームレスな誘導**
    *   セッションを完了した後のユーザーに「余韻」と「次への意欲」を与える体験設計が不足している。

---

## 3. 体験上の「最大の谷」と改善仮説

### 谷：完了後のモチベーションの断絶
*   **事実**: リザルト画面で祝福された後、「セット選択へ戻る」ボタンを押すと、単にセット一覧に戻るだけで、次に何をすべきかのガイドが一切行われない。
*   **なぜ問題か**: ユーザーの「よし、次もがんばるぞ」という学習熱量がその瞬間に急激に冷めてしまい、アプリを閉じる（離脱する）要因になってしまう。
*   **改善仮説**:
    1.  リザルト画面に、次に推奨されるレベルのセット（例: 初級セットA完了後に「初級セットBに挑戦する！」など）へのダイレクト遷移ボタンを設置する。
    2.  「本日の学習ノルマ達成！🌸」のような、1日の目標クリアを称えるストーリー演出を取り入れ、明日また来たくなる余韻を作る。
