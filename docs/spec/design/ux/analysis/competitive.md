## 1. 希望.JPG（Mazii）の観察

ユーザー提示の参照UI（日越辞書アプリ Mazii の単語帳画面）から読み取れる特徴：

| 要素 | 参照UIの特徴 |
|------|-------------|
| 全体トーン | **明るい白基調＋青アクセント**。清潔で見やすい（現状の黒とは真逆） |
| 上部カード | 見出し語を大きく中央に、発音🔊を直下に配置。1語に集中させる |
| リスト表現 | 各語を**白カードで分離**、見出し語＝青、読み/意味＝黒、右上に🔊 |
| メタ情報 | 日付・自分メモ欄・シャッフル・再生ボタンをリスト上部に |
| ページャ | 下部に「1 / 149」「50/Trang(ページ)」＝件数コントロール |
| 主要CTA | 最下部に青の大ボタン「Luyện tập（れんしゅう）」＝行動導線が明確 |

## 2. 観点比較

| 観点 | 参照UI（Mazii） | 現状（本アプリ） | 差分 / 学び |
|------|----------------|-----------------|-----------|
| 第一印象 | 明るい・清潔・親しみ | 暗い・硬派・冷たい | **明るい基調へ寄せる**（ただし青ではなく要望のピンク） |
| リスト表現 | 白カードで1語ずつ分離、発音右上 | 半透明カードで分離済み・バッジ有 | 構造は近い。**配色を明るく＋発音ボタンの視認性UP** |
| フィードバック（正解演出） | （辞書のため学習演出は弱い） | 無 | 参照にも無い → **ここは独自の強みで差別化**（音・祝福） |
| 継続の仕掛け | 弱い（辞書寄り） | 無 | 参照を超える余地。ストリーク等で優位に立てる |
| 情報密度・余白 | 適度な余白・読みやすい | 余白はあるが暗くて沈む | 余白は活かし、**明度を上げて軽やかに** |
| 行動導線 | 最下部に明確なCTA | 学習は常時表示（CTA概念が薄い） | **主要アクションを明快に**（学習の入口/継続ボタン） |

## 3. 取り込む / 避ける

### 取り込む（○）
- **明るい基調**（→ ただし青ではなく **ピンク**。要望＋対象ユーザーに最適化）
- **1語に集中させる上部カード＋発音直下**の型
- **白カードでリストを分離**し、見出し語に色を効かせる
- **明確な主要CTA**（下部固定の行動ボタン）

### 避ける（✕）
- **青のアクセント**（→ 男性的/事務的。ピンク世界観に合わない）
- **辞書的な情報密度**（メモ欄・多数のメタ情報 → 本アプリは学習集中を優先し盛らない）
- **参照の丸写し**（Maziiは辞書、本アプリは"報われる単語帳"。演出で差別化する）

## 4. 差別化の核

参照UI（辞書）は「引く」ためのUI。本アプリは「**続けて・報われる**」ためのUI。
→ 参照から**明るさ・情報整理・CTA明快さ**を借り、
　**正解演出・音・ストリーク・完了祝福・マスコット**という感情設計で明確に上回る。
