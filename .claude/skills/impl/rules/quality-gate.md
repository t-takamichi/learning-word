# Quality Gate — 実装完了チェックリスト

以下を**全て**満たした場合のみ「Phase <N> 実装完了」とする。

## 実装
- [ ] 計画書（`docs/spec/impl/steps/<番号>/plan.md`）の全タスク（T1〜Tn）を実装した
- [ ] 計画外の変更を勝手に加えていない（加えた場合はユーザー承認済み）
- [ ] レイヤー依存方向（`docs/rules/architecture.md`）に反する実装がない

## 品質・規約
- [ ] `tsc --noEmit`（型チェック）が通る
- [ ] `any` の乱用がなく、Hono API は `zValidator` で検証している
- [ ] `docs/rules/` の各規約（TypeScript / Hono / React）に違反していない
- [ ] 既存コードのスタイル・命名に揃っている

## 受け入れ基準
- [ ] 計画書の「Done の定義」チェックリストを全て満たす
- [ ] 設計書のAC（受け入れ基準）を満たす
- [ ] `phase-N.md` の「ゴール」が実際に動作確認できる

## 報告
- [ ] 実装内容と検証結果を正直に報告した（失敗・スキップも明示）

## 判定
- **全項目チェック済み** → Phase 実装完了。次フェーズへ進む
- **未チェック項目あり** → 未完了。該当タスクへ戻って修正する
