# Quality Gate — フェーズ設計完了チェックリスト

以下を**全て**満たした場合のみ「フェーズ設計完了」とし、実装計画フェーズ（impl-plan）へ進む。

## ドキュメント
- [ ] `docs/spec/impl/phase/README.md` が存在し、全フェーズ一覧が書かれている
- [ ] 各フェーズの詳細 `docs/spec/impl/phase/phase-N.md` が存在する
- [ ] フェーズ間の依存関係が Mermaid で図示されている

## 網羅性
- [ ] 設計書（`docs/spec/design/README.md`）の Must have 機能が全てどこかのフェーズに含まれている
- [ ] 各フェーズが対応する設計書へリンクされている（トレーサビリティ）
- [ ] Won't have がフェーズに混入していない

## フェーズ品質
- [ ] Phase 1 が Walking Skeleton（全レイヤーを貫く最小動作）になっている
- [ ] 各フェーズに「完了時点で何が動くか（ゴール）」が明記されている
- [ ] フェーズ順序がレイヤー依存方向・MoSCoW優先度と矛盾しない
- [ ] リスク「高×高」の項目が早いフェーズに配置されている

## 判定
- **全項目チェック済み** → フェーズ設計完了。`/impl-plan [番号]` へ進む
- **未チェック項目あり** → 未完了。該当STEPに戻って補完する
