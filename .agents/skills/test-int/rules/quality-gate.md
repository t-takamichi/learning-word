# Quality Gate — Test Creation & Review

以下のチェックリストをすべてパスした時点で「テスト作成完了」とする。

- [ ] **テスト実装**: 指定対象に対するテストコードが正しく書き下ろされていること。
- [ ] **批判的レビュー**: 否定的なレビュアーによるアドバーサリアルレビューが実行され、アサーションの甘さや境界値漏れが抽出されていること。
- [ ] **ユーザー合意 ＆ ループ**:
  - 否定的なレビュアーの指摘結果がユーザーに明確に共有されていること。
  - 「この指摘・内容で進めてよいか、さらなる指摘や助言があるか」をユーザーに確認し、助言・追加指摘があれば実装エージェントで再修正 ➡️ レビューのループが完了していること。
- [ ] **テストパス**: 作成されたテストが実際に実行され、すべて PASS すること。
