# Quality Gate — UX分析 完了チェックリスト

全項目を満たしたら「分析完了」とし、`/ux-design` へ進む。

## 成果物
- [ ] `analysis/inventory.md`（現状トークンの実値付き）
- [ ] `analysis/heuristics.md`（深刻度付き）
- [ ] `analysis/emotion.md`（感情曲線＋谷の特定）
- [ ] `analysis/competitive.md`（参照UIとの比較・取捨選択）
- [ ] `analysis/findings.md`（優先度付き課題＋改善仮説＋申し送り）

## 内容
- [ ] 全課題が「事実→なぜ問題か→改善仮説」で書かれている
- [ ] 正解フィードバック・継続の仕掛け・完了の祝福について現状が診断されている
- [ ] Quick Win（高インパクト×低コスト）が明示されている
- [ ] 設計フェーズで決めるべき論点が申し送られている

## 判定
- 全✅ → 分析完了。`/ux-design` へ
- 未✅ → 該当STEPへ戻る
