# STEP 4: モチベーション機構・演出・マイクロコピー

**目的**: 「正解したらモチベが上がる」を、瞬間ごとの設計に落とす。

## 実施内容
- **モチベーションループ**: きっかけ→行動→ごほうび→積み重ね の全体像
- **機構の優先度付け**（Must/Should/Could）
  - Must: 正解フィードバック（音×視覚×言葉の3チャンネル）/ 進捗可視化
  - Should: 連続正解コンボ🔥 / 完了の祝福（コンフェッティ＋マスコット）/ デイリー・継続日数
  - Could: XP・レベル・スタンプ / 週間カレンダー
- **正解演出の具体**: トリガー・3チャンネルの内容・タイミング（200ms以内）
- **失敗（Again）の扱い**: 責めない。低刺激・やさしい言葉
- **マイクロコピー**: 命令形→応援形の対応表（旧❌／新✅）
- **各機構のAC**

## 制約
- `prefers-reduced-motion` で簡略化する前提を各演出に付ける
- やりすぎ防止（P4）: 演出は"一手間"まで。完了祝福は2.5秒以内に収束
- Could は「V1では作らない」を明記し過剰実装を避ける
