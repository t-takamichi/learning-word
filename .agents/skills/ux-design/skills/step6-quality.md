# STEP 6: ディズニーPM品質バー・感情曲線

**目的**: 「このレベルのサイトで良いか」をディズニーの思想で判断可能にする。

## 実施内容
- **ディズニー4思想の翻訳**: Plussing / Theming / Emotional Arc / Attention to Detail をこのアプリに適用
- **感情曲線の設計**: 入場→学習（小さな達成の連続）→完了のクライマックス→余韻。谷を作らない
- **現状へのPM判定**: 機能◯／体験✕ のように事実で合否を示す
- **合否チェックリスト**: Delight / Reward / Consistency / Craft / Respect の観点で"ゲストの気持ち"で確認
- **やりすぎ防止（Anti-goal）**: 常時アニメ・鳴りっぱなし・長い祝福・原色過多・盛りすぎを禁止事項化
  - 「引き算された上質さ ＞ 盛り盛りの賑やかさ」

## 出力先
`docs/spec/design/ux/quality-bar.md`

## 注意
- Plussing（一手間を足す）と P4（うるさくしない）を必ず両立させる
- 判定は感想でなく、観点ごとの✅/❌で示す
